# HR4Techies Talent Intelligence Platform (MVP)

## Overview

HR4Techies Advanced People Analytics is a Streamlit-based MVP designed to transform employee data into actionable insights for strategic talent decision-making.

The platform allows users to upload a CSV dataset and explore multiple HR analytics modules through interactive visualizations.

## Key Features

- Performance Analytics
- Strategic Attrition Detection
- Intelligent Role Matching
- Talent Outliers Identification (3D analysis)
- Workforce Archetypes Segmentation

## How it works

1. Upload a CSV file with employee data
2. Select an analysis module
3. Explore visual insights and recommendations
4. Download enriched dataset

## Tech Stack

- Python
- Streamlit
- Pandas / NumPy
- Plotly
